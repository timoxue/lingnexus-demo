位置: config/model_config.json

内容（需填入 API Key）:
[
  {
    "config_name": "qwen-max",
    "api_key": "YOUR_QWEN_API_KEY"      👈 填入
  },
  {
    "config_name": "deepseek",
    "api_key": "YOUR_DEEPSEEK_API_KEY"  👈 填入
  },
  {
    "config_name": "gemini",
    "api_key": "YOUR_GEMINI_API_KEY"    👈 填入
  }
]

获取 API Key:
• Qwen-Max:  https://dashscope.console.aliyun.com/
• DeepSeek:  https://platform.deepseek.com/
• Gemini:    https://aistudio.google.com/

🛠️ 故障排查：

Q1: RDKit 安装失败？
    conda install -c conda-forge rdkit

Q2: API 调用失败？
    检查 config/model_config.json 中的 API Key

Q3: 界面打不开？
    检查端口 7861 是否被占用
